HDR

A massive Squarespace build completed as a team effort, and the most technically intensive work on the project was mine. My contributions span three major builds: the Downtown Directory, powered by CommunityBox — a platform with extremely limited front-end flexibility — which was completely rebuilt visually using custom CSS, JavaScript, and HTML, including a custom internal page navigation system that allows users to move fluidly between directory sections; a full Elfsight events calendar integration, styled with custom CSS to match the brand consistently across multiple page contexts; and the Downtown Dollars commerce flow — a complete MailChimp and Zapier integration that enables visitors to purchase Downtown Dollar credits redeemable at local businesses, with all order management, backend product setup, and email notification flows fully configured. Each of these builds required navigating platform constraints that had no native solution, demanding creative problem-solving and original code at every turn. The result is a site that functions at a level far beyond what any of its underlying platforms were designed to support on their own.
