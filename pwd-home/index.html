PWD Home

A ground-up Squarespace website build — a complete platform migration, not a redesign — that ran concurrently with a full rebrand executed by TDC's graphic design team, including a new logo and the client's evolution from PWD Window and Door to PWD Home. The goal from the start was to build a site that would scale gracefully — one the team could maintain and grow without requiring constant developer involvement. The centerpiece of that strategy is a dynamic content system built on Squarespace's summary blocks, configured with custom tagging, filtering, and parameters so that project spotlights automatically populate across the homepage, service pages, and other key pages site-wide as new content is added. No one has to manually update which projects appear where — the site does it automatically. Beyond the dynamic content system, the site features a custom homepage carousel slider for services and custom mega menus throughout. The result is a clean, modern Squarespace site that honors the old brand's best elements while fully embracing the new direction — and one that was built to keep delivering for PWD Home for years to come.
