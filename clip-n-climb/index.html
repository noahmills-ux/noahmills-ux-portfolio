Clip 'n Climb Website

This was project was designed towards the tail end of the Horizons Edge Website Project. The website is an information page for the Horizons Edge Clip 'n Climb facility. Users can view all of the various climbing walls as well as the rules and policies for the facility. This website is linked in the Horizons Edge site navigation. This website's global navigation features a unique backlink to the horizons edge website in the top right hand corner as well as a global registration button that takes users to the registration page in DaySmart Recreation. I really enjoyed working with the vibrant colors and themes that the Clip 'n Climb global brand utilizes. Super fun project!
