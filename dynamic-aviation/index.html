Campus Movers

A ground-up Squarespace build for a startup client with no prior web presence — an atypical engagement for TDC — that also required managing full oversight of a custom web application built on Zoho Creator by an external development team. On the web app side, I served as the sole point of contact between the client and the developers — fielding all feedback, translating requirements, and keeping the build on schedule while ensuring it met accessibility and technical standards throughout. The website itself was built entirely from scratch with extensive custom HTML, CSS, and JavaScript, featuring a homepage with an advanced animated vertical timeline section and a fully custom interactive cost calculator that estimates move pricing based on user inputs, connected to a conversion-driving pop-up in the lower corner of the screen that funnels visitors directly to the registration form. The project required operating simultaneously as a developer, project manager, and client liaison — and doing all three well enough that a startup with no existing digital infrastructure launched with a polished, conversion-optimized web presence on day one.
