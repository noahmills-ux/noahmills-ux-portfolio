Horizons Edge Website

This was the first full website project I completed after I graduated from JMU in 2022. I took on the project as soon as I started working with Endless Horizons in late February 2023. The website was developed over a 3 month period and launched in May 2023. The project was a redo of the old Horizons Edge Website and the primary goal of the project was to improve the overall User Experience. The website was designed on Wordpress while also utilizing Toolset to create custom fields and layouts to display upcoming events, tournaments, leagues, and sports camps.
