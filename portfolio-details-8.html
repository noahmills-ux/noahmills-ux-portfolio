Project Description

For this project I developed an E-Business proposal for a hypothetical startup business that was looking to launch a new website or mobile application to help carry out its business objectives. I decided to create a website proposal for a new hypothetical drum-business called Carrson Productions. For this project, I did not create an actual website, instead I focused on learning all of the neccesary strategies that are utilized when designing a website for a client. The proposal I created covers all aspects of the website design process including: User Research, SEO Optimization, Production Costs, etc.
