AGC of Greater Florida

This one was a full Squarespace website rebuild tied to a complete organizational rebrand — from AGC of South Florida to AGC of Greater Florida. Every page, layout, and interactive element was designed and built by me alongside an initial brand direction from the design team. The site features a fully custom events calendar that required extensive CSS and JavaScript to overcome Squarespace's native calendar limitations on both desktop and mobile — the platform simply couldn't deliver the display quality and usability the client needed, so I engineered a solution from scratch. The calendar serves as a primary resource for members navigating events and registrations, which link seamlessly into the member portal. I also connected the main site to the member site and updated the member portal itself to reflect the rebrand throughout. The most persistent challenge was making a technically limited platform behave like a custom-built one, while simultaneously managing the complexity of a live rebrand mid-production. The end result is a fully responsive, professionally polished web presence that accurately represents the expanded scope and identity of the newly named organization.
