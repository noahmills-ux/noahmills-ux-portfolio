Project Description

This was my first ever UX Design course. The main project I did in this course was all about User Research and basic prototyping. Throughout the semester we learned how to create user personas, conduct user interviews, sketch wireframes, design low-fidelity prototypes, and how to formulate research questions along with a design strategy. For this project I decided to create a protoype for a new school parking app that would replace the current JMU Parking App. When starting out I asked the design question: how one might better the experience of students trying to locate vacant parking on campus. I chose this design question based off my own experiences with JMU parking, and because JMU parking is a large resource that many students utilize that live both on and off campus. Parking on campus is important to JMU students for these reasons. For the User Research portion of this project I conducted several interviews with JMU students who park on campus regularly. Based off of the data gathered from those interviews, I created two user personas that helped me gain an idea of the user needs for the app I was going to prototype.
