Project Description

This project was the second website I designed using HTML and CSS. This site is a spoof retail site that only sells lemon-related food products. The project requirements included creating a privacy policy statement as well as listing various products and services that are for sale. For this website I also had to optimize the content for mobile and desktop viewing, so the site's design is responsive. This assignment helped me get a lot more comfortable with content organization and web coding.
